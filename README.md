## Overview
This plugin aims to have zero tolerance for payment gateways or real-money transactions. It is designed to provide an in-game economy system for Hytale servers, allowing players to earn, spend, and trade virtual currency without any real-world financial involvement.

## Features
This is a checklist of features that Merchantree will hopefully include:
- [ ] Virtual currency system
- [ ] Player accounts and balances
- [ ] Admin commands for managing economy
- [ ] Player-to-player trading
- [ ] Marketplace system
- [ ] Integration with other Hytale plugins
- [ ] Customizable currency names and symbols
- [ ] Economy statistics and reports

## Installation
1. Download the latest release of Merchantree from the [GitHub releases page](https://github.com/Aoof/hytale-merchantree/releases).
2. Place the downloaded `.jar` file into your Hytale server's `mods` directory.
3. Restart your Hytale server to load the plugin.

## Development
If you wish to contribute to the development of Merchantree, please follow these steps:
1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine.
3. Create a new branch for your feature or bug fix.
4. Copy HytaleServer.jar from your Hytale server directory to the root of the project.
5. Make your changes and commit them with clear messages.
6. Push your changes to your forked repository.
7. Open a pull request to the main Merchantree repository.
